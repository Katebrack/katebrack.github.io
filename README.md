# Kate Brack — Personal Site

A simple, fast, single-page personal website for landing freelance/advisory
marketing clients. No build step — just plain HTML, CSS, and a little JavaScript.

## Files
- `index.html` — all the content/copy
- `styles.css` — the look & feel (colors live in the `:root` block at the top)
- `main.js` — mobile menu, scroll effects, footer year

## Edit it
Open `index.html` in any editor and change the text. Refresh your browser to see it.

## Preview locally
```
python3 -m http.server 4173 --directory .
```
Then open http://localhost:4173

## Things to update
- **Booking link:** in `index.html`, search for `calendly.com/katebrack` and swap in your real link.
- **Email:** search for `katebrack@gmail.com` to change the contact address.
